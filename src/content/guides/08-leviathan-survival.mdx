<Callout type="danger" title="核心规则">
**所有利维坦级生物完全无敌**，无法击杀。唯一选择是规避、逃跑、利用地形遮蔽。带武器去送死是新人最常见的错误。
</Callout>

## Collector Leviathan（采集者）

### 识别方式

| 信号 | 描述 |
|------|------|
| **声纳脉冲** | 多段节奏的声纳声 = 附近有 Collector |
| **生物发光** | 身体密集发光点反映大脑活动 |
| **体型** | 巨大头足类，装甲喙 + 四条触手 |

### 行为模式

- 领地性强，在固定区域巡逻
- 对小型潜航器（Tadpole）和基地感兴趣
- 会用触手撬开硬壳目标

### 规避策略

| 情况 | 应对 |
|------|------|
| 听到声纳 | 立即停止移动，关闭 Tadpole 灯光 |
| 被发现 | 全速逃离，利用岩石遮蔽 |
| 在基地附近 | 进入基地等待它离开 |
| 需要穿越领地 | 贴海底地形走，避开开阔水域 |

<Callout type="tip" title="灯光是关键">
被 Collector 锁定时**关闭 Tadpole 灯光**可以显著降低仇恨。开车进它领地永远关灯。
</Callout>

## Void Leviathan（虚空）

### 识别方式

- 进入虚空区域（地壳架外的开阔水域）
- 群体出现：雄性成群，由一只雌性领导
- 镰刀状鳍的独特轮廓

### 行为模式

- 攻击任何进入虚空的物体
- 群体协同攻击
- **不可逃脱**（设计上就是世界边界惩罚）

### 规避策略

<Callout type="danger" title="虚空 = 死亡">
**不要进入虚空。** 就这么简单。
</Callout>

如果你不小心游到了地图边缘：

1. 立即调头
2. 全速返回地壳架
3. 已经被攻击 → 大概率死亡，接受现实

## 通用利维坦生存技巧

<Callout type="step" title="保命五条">
1. **听声音**：每种利维坦有独特音效，调大音量提前预警  
2. **用信标标记**：在巡逻边界放信标，提醒自己和队友  
3. **不要停留在开阔水域**：永远贴着地形走  
4. **关 Tadpole 灯光**：被追击时关灯可降低仇恨  
5. **合作模式分工**：一人引开，其他人趁机通过
</Callout>
