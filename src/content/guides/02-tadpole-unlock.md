## 为什么 Tadpole 是最重要的里程碑

- 无限氧气（只要有电）
- 250m 基础深度
- 移动速度翻倍
- 内置照明
- 可携带储物柜

## 解锁步骤

1. **找到 Tadpole Pens**：在 Coral Gardens 深处的设施
2. **获取密码**：探索周围的 PDA 日志获取 Key Code
3. **进入设施**：扫描内部的 Tadpole 碎片
4. **建造 Mobile Vehicle Bay**：需要先解锁此蓝图
5. **制造 Tadpole**：钛 ×3 + 玻璃 ×1 + 铜 ×2

## 升级优先级

解锁后立即制造 **深度模块 1**，否则 250m 不够探索中深海。

## 合作模式技巧

- 队友可以抓住 Tadpole 侧把手搭便车
- 一人驾驶一人扫描效率最高
- 共享一台 Tadpole 比每人造一台更省资源（前期）
