<Callout type="danger" title="核心要点">
密码不在 Tadpole Pens 里面！在大约 300 米外的深海洞穴中找 Zip 的 Black Box。
</Callout>

## 完整步骤

### 第一步：找到 Tadpole Pens

- 从 Lifepod 向**东**约 **690 米**
- 位于 Lava Vents（熔岩喷口）区域
- NOA 会给你信号标记「Investigate Tadpole Pens」

<Callout type="tip" title="前置条件">
需要先找到 Wander Blackbox 并治愈 Angel Comb，NOA 才会触发此任务。
</Callout>

### 第二步：进入设施

1. 进入 Tadpole Pens 建筑
2. 找到**红色按钮面板**
3. 按下按钮 → 提示需要密码
4. **不要在这里浪费时间找密码**

### 第三步：前往 Zip Black Box

<Callout type="step" title="关键路线">
从 Tadpole Pens 出发 → 向西南游 210 米 → 找到悬崖边的外星装置 → 从左侧大缝隙下潜 → 进入大洞穴 → 找到紫色根部上的 Black Box
</Callout>

详细路线：

1. 从 Tadpole Pens 入口出发，向**西南方向**游
2. 沿途注意 **Oxygen Tunic**（蓝色冒泡植物）补充氧气
3. 看到悬崖边的**圆形外星装置** → 从旁边的大缝隙下潜
4. 进入隧道，通向一个**中央有火焰珊瑚的大洞穴**
5. 洞穴北偏东侧有 Oxygen Tunic → 旁边有个洞口
6. 一路下潜到底部 → **紫色大根部上就是 Zip 的 Black Box**
7. 互动获取 **Kurul Level Access Key**

<Callout type="warning" title="生存准备">
洞穴深度 310 米，出发前确保：携带 Air Bladder（气囊）、满电扫描枪、食物和水。Hot Caves 区域有热量伤害。
</Callout>

### 第四步：返回解锁

拿到密码后返回 Tadpole Pens → 再次按红色按钮 → 门打开 → 进入内部有**可呼吸空气的房间**和 NOA 终端。

## 常见问题

**Q：按了红色按钮但什么都没发生？**

回 Lifepod 检查 NOA 是否有未读消息。需要先触发任务。

**Q：洞穴入口在哪？找不到。**

从 Tadpole Pens 出来后面朝西南（指南针确认），贴海底游约 210 米。找悬崖边缘的圆形外星装置，旁边就是入口。

**Q：拿到密码后门还是打不开？**

确认你互动的是 Zip 的 Black Box（橙色外观），不是普通补给箱。
