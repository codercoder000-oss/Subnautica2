## 中期目标清单

按推荐顺序：

1. ✅ 强化合金锭研究
2. ✅ 旋风者蓝图集齐
3. ✅ 旋风者建造（消耗大量资源）
4. ✅ 深海基地建设
5. ✅ 失落之河探索

## 资源储备建议

进入中期前确保库存：

| 材料 | 数量 |
|------|------|
| 钛锭 | 5+ |
| 强化合金锭 | 3+ |
| 锂 | 10+ |
| 钻石 | 5+ |
| 红宝石 | 3+ |

建议**先建中期资源基地**（水母蘑菇洞穴边缘），就近采集。

## 旋风者优先级

旋风者是中期主力，但**不是必须立刻建造**。

- 资源充足 → 建
- 资源紧张 → 先升级海蛾号到深度 3，足够探索大堡礁

## 深海首次冒险

第一次进失落之河之前：

- 旋风者深度模块 1
- 修复工具 + 修复多块**预制壁板**
- 至少 2 块满电动力电池备用
- 急救包 ×3
