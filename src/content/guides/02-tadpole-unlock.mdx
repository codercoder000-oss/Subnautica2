<Callout type="danger" title="一句话答案">
Tadpole 是游戏最重要的里程碑。解锁它需要：找到 Tadpole Pens → 获取密码 → 扫描碎片 → 建造 Mobile Vehicle Bay → 制造 Tadpole。
</Callout>

## 为什么 Tadpole 是最重要的里程碑

<Callout type="info" title="Tadpole 给你什么">
- **无限氧气**（只要有电）
- **250m 基础深度**
- **移动速度翻倍**
- **内置照明**
- **可携带储物柜**
</Callout>

没有 Tadpole，你只能在浅海 100m 内活动。有了它，整个中深海世界向你打开。

## 解锁步骤

<Callout type="step" title="完整流程（5 步）">
1. **找到 Tadpole Pens**：在 Coral Gardens 深处的设施（Lifepod 东方约 690m）
2. **获取密码**：探索周围找到 Zip 的 Black Box 获取 Key Code
3. **进入设施**：扫描内部的 Tadpole 碎片（需要扫描 3 个）
4. **建造 Mobile Vehicle Bay**：需要先解锁此蓝图（扫描碎片）
5. **制造 Tadpole**：钛 ×3 + 玻璃 ×1 + 铜 ×2
</Callout>

<Callout type="warning" title="密码不在 Tadpole Pens 里面">
最常见的卡关点：密码在 300 米外的深海洞穴里，不是在设施内部找。详见 [Tadpole Pens 密码攻略](/guides/05-tadpole-pens-keycode/)。
</Callout>

## 升级优先级

<Callout type="tip" title="解锁后第一件事">
立即制造 **深度模块 1**，否则 250m 基础深度不够探索中深海。配方：钛锭 ×1 + 玻璃 ×1。
</Callout>

## 合作模式技巧

- 队友可以抓住 Tadpole 侧把手**搭便车**
- 一人驾驶一人扫描效率最高
- 共享一台 Tadpole 比每人造一台更省资源（前期）
