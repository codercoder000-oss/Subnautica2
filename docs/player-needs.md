# 玩家诉求分析（来自 Steam 论坛 / Reddit / 攻略站评论）

---

## 一、玩家最高频搜索/提问（按热度排序）

### 1. Tadpole Pens 密码在哪？（最热门问题）
- 密码不在 Tadpole Pens 本身，而是在 300 米外的深海洞穴里
- 需要找到 Zip 的 Black Box
- 路线：从 Tadpole Pens 向西南方向，下潜到 236m 深的洞穴
- **我们需要做**：写一篇详细的 Tadpole Pens 密码攻略，带坐标和路线图

### 2. 资源在哪找？（铜、金、钛）
- 二代的 Raw Ore 系统让很多玩家困惑
- 不再有独立矿种，一个 Raw Ore 随机出三种
- 洞穴密度更高
- **我们需要做**：资源获取指南，解释 Raw Ore 机制

### 3. 多人联机问题
- 连接失败、掉线、跨平台问题
- 防火墙设置
- 主机玩家离线世界关闭
- **我们需要做**：多人联机故障排除指南

### 4. 性能优化
- 8GB 显存 GPU 在密集区域掉帧
- UE5 崩溃问题
- 加载时间优化
- **我们需要做**：性能优化设置指南

### 5. Biomod / Adaptations 怎么获取
- 目前已知 4 种 Adaptations：Pressure Tolerance, Digestion, Heat Tolerance, Axum Vision
- 需要 Bioscanner 工具
- Biolab 位置
- **我们需要做**：完整 Biomod/Adaptations 指南

### 6. 利维坦不可击杀怎么办
- 二代设计决策：所有利维坦不可击杀
- 玩家需要学会规避而不是战斗
- **我们需要做**：利维坦规避策略专题

## 二、玩家情绪热点

### 正面
- 画面震撼（UE5 效果）
- 合作模式好评
- 生物 AI 行为真实
- 探索感强

### 负面
- **EULA 争议**：隐私条款过于激进，大量差评
- **内容量少**：EA 阶段地图小，几小时就探索完
- **过度引导**：Kotaku 批评「游戏从不安静，不停告诉你该做什么」
- **性能问题**：密集区域掉帧
